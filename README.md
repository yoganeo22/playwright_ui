# UI Automation using Playwright

## Installations
Install:
1. Nodejs
2. Git Bash
3. VS Code (extension `Playwright for VS Code`)

## Tests
1. api.spec.ts
2. googlePageTest.spec.ts
    - Failed login
    - Search Google
3. playwrightPageTest.spec.ts
    - Toggle Button
    - Pop-up window
    - Drop down list
    - Icon

## Create New Project
Run command:
```
npm init playwright@latest
```
